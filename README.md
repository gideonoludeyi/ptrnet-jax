# Reinforcement Learning for the Vehicle Routing Problem (JAX/Flax)

This project implements a deep reinforcement learning approach to solve the Vehicle Routing Problem (VRP) using JAX and Flax. It builds upon the methodologies proposed in recent research, specifically employing an Actor-Critic model with a Pointer Network architecture.

See `examples/` for how to use the model.

## Reference

This project is inspired by and adapts code from the following PyTorch implementation:
- [mveres01/pytorch-drl4vrp](https://github.com/mveres01/pytorch-drl4vrp.git)

## Original Paper

The core methodology implemented in this project is based on the research paper by Nazari et al.[^1]


[^1]: Nazari, M., Oroojlooy, A., Snyder, L.V., & Takác, M. (2018). Reinforcement Learning for Solving the Vehicle Routing Problem. *Neural Information Processing Systems*.
